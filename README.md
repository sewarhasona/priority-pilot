סקירה כללית
אפליקציית Web לניהול ומעקב אחר פגישות למידה אישיות, המאפשרת למשתמשים לתעד את זמן הלמידה שלהם, את הנושאים הנלמדים ולנהל היסטוריית למידה מאורגנת.

איזו בעיה הפרויקט פותר
הפרויקט פותר את הקושי במעקב אחרי התקדמות אקדמית לאורך זמן, מונע אובדן מידע לגבי חומרים שנלמדו, ומסייע לסטודנטים לשמור על ריכוז והתמדה בתהליך הלמידה ללא צורך במעקב ידני מסורבל.

קהל היעד
המוצר מיועד לסטודנטים וסטודנטיות המעוניינים בארגון זמן יעיל יותר, שזקוקים למערכת פשוטה ונגישה כדי לתעד פגישות למידה בזמן אמת, בין אם בבית או בקמפוס.

מתחרים ובידול

מתחרים קיימים: מעקב ידני בעזרת קבצי Excel, מחברות פיזיות, או התכתבויות בוואטסאפ לתיעוד נושאים.
הבידול שלנו: המערכת מציעה פתרון Full-Stack מובנה המבוסס על React ו-Supabase, מה שמאפשר נגישות מכל מכשיר, אבטחת נתונים עם RLS (Row Level Security), וחוויית משתמש ממוקדת מטרה ללא העומס הקיים בניהול ידני או בכלי ניהול פרויקטים מורכבים מדי.
